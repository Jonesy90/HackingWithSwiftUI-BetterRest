# Hacking With SwftUI - Better Rest

## Description:
This app introduced me to CoreML (Machine Learning), this allowed me to write code that makes predictions about new data based on previous data it has seen.

First, I gave it some raw data which it used as training data, the results are then used in the app to make estimated about new data (all on device).

This app takes in some data (wake up time, how many hours sleep and cups of coffee drunk), with this information it returns a estimated bed time.

## What I Learned
1. CoreML
2. Create ML
3. Dates
